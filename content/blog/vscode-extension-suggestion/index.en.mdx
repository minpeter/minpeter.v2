## Paste Image from local pc

![](/images/3e3bba50-8b7f-4a2d-87f5-bda7baa7d2ef-2022-01-10-00-21-29.png)
This is truly revolutionary!
Essential when writing posts for static blogs like Jekyll or Hugo!
Simply insert images with Ctrl + Alt + V after taking a screenshot.
If only I had known about this feature, my blog posts would have been a bit prettier...

## GitHub Copilot

![](/images/001bc390-f94e-41c5-bf55-8da7f0f6ba44-2022-01-10-00-22-22.png)
When learning a new framework or language,
it perfectly compensates for my slow typing speed when entering unfamiliar syntax.
Everything is great, but you need to be selected as a beta tester to use it,
and there are some serious copyright issues.
